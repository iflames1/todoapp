# Todo List App

## Features:
- Add a new task to the list.
- Mark an existing task as Done.
- Delete a task.
- Filter tasks by status (Done/Not done).

## Upcoming Feature:
- Allow login and create account so as to save Tasks.
- Assign Title to a task.
- Edit existing task.
- Push notification/reminder for task.
- Add due date for each task.
- Add priority to tasks.
- Add task reminder.
- Dark Mode Theme.

## Technology Used:
- React JS - Library
- CSS - Styling
